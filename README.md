# APE 2: Estructuras de Selección - Algoritmos y Lógica de Programación
Este repositorio contiene la resolución de la APE Nro. 02, correspondiente a la asignatura de **Algoritmos y Lógica de Programación** de la carrera de **Software** en la Universidad Técnica de Ambato.

## 📌 Información General
* **Institución:** Universidad Técnica de Ambato (UTA).
* **Facultad:** Ingeniería en Sistemas, Electrónica e Industrial.
* **Ciclo Académico:** Enero 2026 - Julio 2026.
* **Nivel:** Primero.

## 🎯 Objetivo
Utilizar estructuras de selección en el análisis y diseño de algoritmos para resolver problemas de lógica de programación mediante la implementación en C++.

### 1. Control de Hora
Programa que permite el ingreso de horas, minutos y segundos con validación de rangos. Permite al usuario visualizar el tiempo registrado y decidir si desea modificarlo antes de finalizar.

### 2. Sistema de Cuentas Bancarias
Simulación de una gestión bancaria para un cliente (identificado por DNI) con las siguientes funcionalidades:
* Creación de hasta 3 cuentas con saldo inicial.
* Menú interactivo para:
    * Ver atributos de cuenta.
    * Realizar retiros (enviar dinero).
    * Realizar depósitos (recibir dinero).
    * Transferencias entre las cuentas del mismo usuario.

### 3. Gestión de "Jean" en Fábrica/Tienda
Implementación de lógica orientada a objetos (clases) para gestionar un artículo de ropa:
* **Atributos:** Código, color, talla, estado de tela, humedad, cantidad de botones y teñidos.
* **Métodos:** * `Lavar()`: Disminuye la cantidad de teñidos del jean.
    * `Secar()`: Reduce el porcentaje de humedad.
    * `MostrarDatos()`: Presenta el estado actual del objeto.
